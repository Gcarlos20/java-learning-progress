# Sistema Bancario en Java

Proyecto de práctica en Java que simula un sistema bancario básico usando
Programación Orientada a Objetos y ArrayList.

## Funcionalidades
- Crear cuentas bancarias
- Depositar y retirar dinero
- Activar y desactivar cuentas
- Listar cuentas activas e inactivas

## Tecnologías
- Java
- VS Code

## Estructura del proyecto
- models → entidades del sistema
- service → lógica del negocio
- utils → menú e interacción con el usuario
- main → punto de entrada
